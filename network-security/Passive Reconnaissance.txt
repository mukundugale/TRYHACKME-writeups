Passive Reconnaissance :
       In that, you rely on publicly available knowledge , it is the knowledge where you get from publicly available resources without engaging with the target.
       
       for ex : you visit facebook company website and search for specific employee details 
       
WHOIS : 
     whois is request and response protocol that follows RFC 3912 specification. whois server listen on tcp port 43 for incoming requests.
     
     command : whois <DOMAIN-NAME>
     
NSLOOKUP :
     nslookup stand for name server look up and it is used to find ipv4 address of domain name.
     
     command : nslookup <DOMAIN-NAME>
     
     you can use these also
     
     command : nslookup OPTIONS DOMAIN-NAME SERVER
     in that options are A for ipv4 address and AAAA for ipv6 address
     -server is for to query the dns servers for ex cloudflare offers 1.1.1.1 and 1.0.0.1 ,google offers 8.8.8.8 and 8.4.4.8 & quad9 offers 9.9.9.9 and 149.112.112.112 .
     
     for instance you command : nslookup -type=A tryhackme.com 1.1.1.1 
     
     for collect info about mail servers then use : nslookup -type=MX tryhackme.com 1.1.1.1 
     
DIG :
     DIG stand for Domain Information Groper it is an another DNS query 
     
     syntax command : dig @server domain-name type 
     
     for ex : dig @1.1.1.1 thmlabs.com TXT
